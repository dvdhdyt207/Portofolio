Hello! I’m Devid Hidayat, a Computer Science graduate with a strong passion for data-oriented careers such as Data Analyst, Data Engineer, and Data Scientist.

I have hands-on experience in analyzing and visualizing data using Python (NumPy, Pandas, Matplotlib, Seaborn), and in building ETL workflows using Pentaho Data Integration with MySQL. I’m also proficient in PHP and Java, and have a solid foundation in data modeling (Star Schema), SQL, and Power BI.

During my internship at Liki Tea Plantation (PT Mitra Kerinci), I collected, cleaned, and analyzed sales data using Excel and applied K-Means Clustering to uncover data patterns that support strategic decisions.

I also developed a Decision Support System for teacher performance evaluation using the MAUT (Multi-Attribute Utility Theory) method, implemented with PHP and MySQL. Additionally, I built a data warehouse with star schema, performed ETL using Pentaho, and visualized the result using Microsoft Power BI.

I’m certified in Data Analysis with Python by freeCodeCamp, and I enjoy working on real-world projects that require structured problem-solving and strong analytical thinking.

With a deep interest in learning and growing in this field, I’m excited to contribute to meaningful, data-driven projects and collaborate in a dynamic team environment.
